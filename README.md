# AI Compute-Dollar Systemic Risk Dashboard

一个基于 Streamlit 的 AI 金融风险监控系统，用于跟踪：

- AI 泡沫风险
- 算力美元（Compute-dollar）体系
- AI 基建金融化
- 数据中心风险
- AI CapEx 与收入背离
- 美元流动性压力
- AI 次贷危机早期信号

整个项目的目标是：

建立一个类似“次贷危机监控面板”的 AI 系统性风险雷达。
